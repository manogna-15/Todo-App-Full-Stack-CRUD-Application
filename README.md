# Todo-App-Full-Stack-CRUD-Application

## 📌 Overview
The Todo App is a full-stack web application that allows users to manage daily tasks efficiently. Users can create, view, update, and delete todos with real-time UI updates. The project demonstrates full-stack integration using React.js for the frontend and Spring Boot for the backend with MySQL as the database.

## 🎯 Features
- Create new todo tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Real-time UI updates

## 🏗️ Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Spring Boot, Spring Web, Spring Data JPA
- Database: MySQL
- Tools: Maven, VS Code, IntelliJ IDEA, Postman

## 🔄 CRUD Operations
- Create → Add new todo
- Read → Fetch all todos
- Update → Edit or toggle task status
- Delete → Remove task permanently

## 🌐 API Endpoints
- GET /todos → Get all todos
- POST /todos → Create todo
- PUT /todos/{id} → Update todo
- DELETE /todos/{id} → Delete todo

## 🚀 Project Flow
User → React UI → Spring Boot API → MySQL Database → Response → UI Update

## 🏁 Conclusion
This project demonstrates full-stack development skills including REST API design, database integration, and modern frontend development using React.
