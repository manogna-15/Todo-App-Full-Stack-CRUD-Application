# Todo-App-Full-Stack-CRUD-Application

## Overview
The Todo App is a full-stack web application that allows users to manage daily tasks efficiently. Users can create, view, update, and delete todos with real-time UI updates. The project demonstrates full-stack integration using React.js for the frontend and Spring Boot for the backend with MySQL as the database.

## Features
- Create Todo
- View Todos
- Update Todo
- Delete Todo
- Real-time UI updates

## Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL

## API Endpoints
- GET /todos
- POST /todos
- PUT /todos/{id}
- DELETE /todos/{id}

## Flow
React → Spring Boot → MySQL → Response → UI Update
